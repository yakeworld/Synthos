## 架构

**Synthos** 是一个可计算的、协作的、持续进化的公共认知操作系统。

```
[1] 知识获取 → [2] 知识提取 → [3] 关联发现 → [4] 观点生成 → [5] 论证表达
                                                              ↓
                                                      [6] 观点验证 ↺
```

所有原子均为**纯 SKILL.md 文件**，Agent 通过加载技能来读取 → 推理执行 → 保存JSON。零 Python 脚本，零编排引擎。

1. Agent 加载原子 SKILL.md → 执行任务 → 输出 JSON

4. 结果写入 evolution-state.json
