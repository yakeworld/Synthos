# 智能体建设说明书

## 智能体名称
Synthos — 自主进化学术科研平台

---

## 概述

### 开发背景

在医学研究领域，信息爆炸式增长与科研效率瓶颈之间的矛盾日益突出。医学研究者每天面临海量文献、复杂数据、多维证据的整合需求，传统的文献综述、假设生成、论证构建等科研环节仍然大量依赖人工，效率低下且质量参差不齐。与此同时，大语言模型和智能体技术在科研场景中的应用仍处于初级阶段——多数工具仅能提供单一的文献检索或文本生成能力，缺乏系统性的科研方法论支撑，难以形成从"知识获取"到"知识验证"的完整闭环。

《教育强国建设规划纲要（2024—2035年）》明确提出"推动人工智能技术在教育领域的深度应用"，《关于加快推进教育数字化的意见》进一步强调"以人工智能重塑教育形态"。在医学教育领域，如何将AI技术深度融入科研流程与教学体系、提升医学研究者的数字素养和科研创新能力，成为亟待解决的核心问题。

在此背景下，**Synthos** 应运而生。它是一个基于原创"Synthos八维认知框架"的自主进化AI科研平台，将科研全流程原子化为6个认知单元（知识获取、知识提取、关联发现、观点生成、论证表达、观点验证），通过有向无环图（DAG）架构实现认知单元的有机协同，并依托持续进化引擎在无人值守状态下自动优化系统性能。

Synthos 同时深度整合了"**医疗数据智能分析**"教学项目——一个包含73个知识来源（涵盖AIGC教学方法论、CRISP-DM医学预测模型方法论、完整教案体系、25+医学数据集、参考著作、学术论文）、覆盖Python科学计算→数据预处理→ML建模→可信度验证→AI论文写作→科研能力培养完整链条的研究生课程资源库。Synthos 将该教学项目作为核心应用载体，使"科研工具"与"教学平台"形成闭环。

### 拟解决的问题

Synthos 致力于解决医学科研与教学场景中的四大核心问题：

1. **文献检索效率瓶颈**: 传统文献检索分散在多个数据库（PubMed、Semantic Scholar、Crossref、OpenAlex），研究者需逐一查询、手动去重、逐一阅读摘要，耗时巨大。Synthos 通过多源并行检索和结构化提取，将"数小时文献调研"压缩至"分钟级知识聚合"。

2. **假设生成与模型构建缺乏系统方法**: 多数科研工具仅提供数据或文本处理能力，缺乏对"如何形成可验证研究假设、构建可信预测模型"的系统方法论支撑。Synthos 基于贝叶斯思维和第一性原理，在知识获取→知识提取→关联发现→观点生成的完整链条上，系统化地生成、评估、排序研究假设，并结合CRISP-DM方法论构建可验证的医学预测模型。

3. **论证过程与论文写作缺乏质量保障**: 论文撰写和论证构建过程中，缺乏自动化质量检查机制。Synthos 引入证伪主义和自由能原理，通过观点验证原子主动寻找反证、评估理论鲁棒性，显著降低"确认偏误"风险。结合TRIPOD+AI报告标准，自动生成符合规范的临床预测模型报告。

4. **科研工具碎片化与教学脱节**: 研究者通常需要组合使用数十种工具（文献管理、文本分析、统计软件、写作工具），且科研工具与教学工具各自独立。Synthos 将这些能力整合为统一的认知原子架构，通过任务路由器自动选择最短执行路径；同时，将73个教学来源整合为可被智能体直接调用的知识资源，实现"科研工具+教学平台"的一体化。

### 核心功能

Synthos 的核心功能模块由6个认知原子、1个任务路由器和2个应用层构成：

**认知原子（6个独立认知单元）**：
- **知识获取**（Knowledge Acquisition）：多源论文检索 + 摘要提取 + 全文下载，支持 PubMed、Semantic Scholar、Crossref、OpenAlex 四大权威数据库；同时整合教学项目73个来源（PDF文献、教案、数据集描述、方法论文档）
- **知识提取**（Knowledge Extraction）：从论文和教学资料中提取结构化知识——方法、数据、结论、局限性，构建知识图谱；从医学数据集中提取关键特征
- **关联发现**（Association Discovery）：识别文献间的矛盾、补充、演进关系和知识空白，输出关联图谱；识别数据集间的关联与互补关系
- **观点生成**（Hypothesis Generation）：基于已有知识和关联分析，生成可验证的研究假设，附带合理性评分；结合CRISP-DM方法论，生成医学预测模型的构建方案
- **论证表达**（Argument Expression）：将假设转化为结构化论文章节，构建论据链，自动匹配参考文献；结合TRIPOD+AI标准生成临床预测模型报告
- **观点验证**（Viewpoint Verification）：主动寻找反方观点，进行证伪检验和鲁棒性测试，输出置信度评估；结合CRISP-DM评估阶段和概率校准技术验证模型可信度

**任务路由器**（Task Router）：
- 基于奥卡姆剃刀原则，自动判断任务复杂度
- 选择最短认知原子执行路径（1-6个原子）
- 避免不必要的认知步骤，提升执行效率

**自主进化引擎**（Evolution Engine）：
- 持续质量评估：定期评估各认知原子的质量分数
- 跨领域技能吸收：通过语义匹配发现和吸收外部优秀技能
- 认知偏差检测：识别系统性偏见，触发补充检索
- 参数优化：无人值守状态下自动优化各原子参数

**教学应用层**（Teaching Application Layer）：
- 将73个教学来源整合为可直接调用的知识资源库
- 支持"智能体辅助教学"模式：教师/学生通过自然语言提问获取教学指导
- 覆盖完整课程链条：Python基础→数据收集→特征选择→ML建模→可信度验证→AI论文写作→科研能力培养
- 核心教学方法论支撑：AIGC辅助教学、CRISP-DM方法论、TRIPOD+AI标准、可解释AI、概率校准

### 技术路线

Synthos 基于 Hermes Agent 平台构建，核心技术栈如下：

1. **智能体平台层**: 基于 Hermes Agent 的自主智能体框架，支持多模型协同、技能组合、状态持久化
2. **认知原子层**: 6个独立认知单元，每个原子为独立 SKILL.md 文件，定义清晰输入输出接口，DAG 依赖无循环
3. **编排层**: 任务路由器根据任务类型自动选择最短执行路径
4. **进化层**: 持续质量评估 + 偏差检测 + 技能吸收 + 参数优化的Agent原生进化闭环
5. **教学应用层**: 73个教学来源整合 + AIGC辅助教学接口 + 课程资源管理
6. **数据层**: 演化状态（evolution-state.json）、教学知识库（73 sources）、原子输出JSON、信任数据（trust_db.json）

技术架构严格遵循 Synthos 八维认知框架：
- **第一性原理**：从基本事实出发，逐层构建知识体系
- **系统思维**：整体视角理解知识关系
- **贝叶斯思维**：基于证据不断更新信念
- **类比思维**：跨领域知识迁移
- **奥卡姆剃刀**：优先最简解释，最短路径执行
- **证伪主义**：主动寻找反证
- **模型依赖实在论**：多视角分析
- **自由能原理**：最小化预测误差，自组织系统

### 创新点

1. **原创认知操作系统**：Synthos 八维认知框架将科学哲学核心理念（第一性原理、证伪主义、贝叶斯思维等）转化为可计算、可执行的认知原子架构，实现了科学方法论的"代码化"。

2. **科研+教学双轮驱动**：Synthos 不仅是科研工具，更是教学平台。通过整合73个教学来源，将CRISP-DM方法论、TRIPOD+AI标准、AIGC辅助教学方法论深度融入智能体能力，实现了"科研工具"与"教学平台"的一体化，构建了完整教学体系与AI智能体深度集成的医学教育平台。

3. **持续进化机制**：区别于静态科研工具，Synthos 具备自主进化能力——通过技能质量评估、跨领域技能吸收、认知偏差检测，在无人值守状态下持续优化系统性能。

4. **质量保障体系**：完整的技能测试、验证、淘汰机制，确保每个认知原子的输出质量。所有原子通过 Agent Skills 规范认证。

5. **奥卡姆剃刀编排**：任务路由器不是强制"全链执行"，而是根据任务复杂度动态选择最短原子路径——简单检索仅用1个原子，完整综述用6个原子，避免认知资源浪费。

6. **CRISP-DM 全流程映射**：6个认知原子天然映射 CRISP-DM 六阶段（业务理解→数据理解→数据准备→建模→评估→部署），使智能体输出的每个结果都具有方法论可追溯性。

## 应用成效

### 实际应用场景

Synthos 核心应用场景包括：

**科研场景**：
1. **NSFC 基金申请支持**：为课题提供完整的文献综述、假设生成、论证构建服务，将基金申请书文献综述部分的撰写时间从 2-3 周缩短至 3-5 天。

2. **研究领域综述**：自动生成领域的最新研究综述，整合 PubMed、Semantic Scholar 等多源数据，识别知识空白和研究趋势。

3. **前庭障碍与眼科交叉研究**：支持多医学领域文献的跨领域关联发现，促进眼动追踪技术在前庭障碍诊断中的应用。

4. **临床预测模型构建**：基于乳腺癌、糖尿病、心脏病等数据集，运用 Synthos 的知识获取→知识提取→观点生成→观点验证完整链条，构建可验证的医学预测模型，输出符合 TRIPOD+AI 标准的分析报告。

**教学场景**：
5. **研究生 AI 辅助科研能力培养**：通过整合73个教学来源，为研究生提供从"零基础"到"独立发表"的完整训练路径。学生通过自然语言提问获取指导，Synthos 自动匹配相关教案、数据集、方法论文档和学术论文，实现个性化学习。

6. **AIGC 辅助教学实践**：教师使用 Synthos 整合的 AIGC 教学方法论和教案体系，重构课程教学流程。从传统的"教师讲授+学生模仿"模式，转变为"AI辅助+问题导向+项目驱动"的新模式，学生自主学习效率和科研产出显著提升。

7. **CRISP-DM 方法论教学**：将 CRISP-DM 六阶段流程映射到 Synthos 的6个认知原子，为学生提供了"理论→实践→工具→方法论"四位一体的学习路径。学生在完成真实临床数据集分析的同时，系统掌握可信赖医疗AI的方法论。
