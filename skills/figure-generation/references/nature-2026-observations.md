# 2026 Nature 页面观察

真实Nature 2026论文的图表原型分析。

## 原型1: 示意图主导复合型

- 示意图占45-60%图高
- 支撑面板使用与示意图相同的物理/材料色板
- 缩放标注使用统一的强调风格（如虚线红色轮廓）
- 至少一个支撑面板为真实照片或实验快照

## 原型2: 暗底图像板

- 黑底仅用于图像板区域，非全页
- 灰度背景 + 1-2个荧光通道（常用青和品红）
- 裁剪、比例尺、视图框几何一致
- 白色比例尺和白色间距使打印/导出后仍可读

荧光通道推荐：
```python
CYAN = "#22D7E6"
MAGENTA = "#FF2AD4"
GREY_CONTEXT = "#B8B8B8"
```

## 原型3: 临床三联画

- 上行：折线/纵向汇总，共享图例条带在上
- 中行：森林图风格效应量 + 虚线垂直参考线 + 浅色类别条带
- 下行：紧凑汇总柱状图（二元或百分比）
- 列语义平行：第一列是ABR，后续列复用相同行逻辑

## 原型4: 密集分类物理面板

- 直接标签区域（当类别多且语义独立时）
- 邻居填充明度接近时使用hatch或纹理叠加
- 完全相同的轴限制和面板几何跨网格复用

## 原型5: 非对称混合模态图

- 一个圆形/示意图面板占网格面积的视觉中心
- 小面板按证据值排序大小
- 不等分面板尺寸不是问题
