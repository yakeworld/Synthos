# Figure Generation Design Theory

吸收自nature-figure（来源：[figures4papers](https://github.com/ChenLiu-1996/figures4papers) 仓库，发表于 *Nature Machine Intelligence* 及顶级ML/Bioinfo期刊）。

## 1) 字体

- **标准**: `font.family='sans-serif'`, `font.sans-serif=['Arial','DejaVu Sans','Liberation Sans']`
- **SVG/PDF可编辑文本**: 必须 `svg.fonttype='none'`
- **LaTeX数学标签**: `text.usetex=True`（仅当LaTeX已安装）

字体尺寸层级：
| 上下文 | font.size |
|--------|-----------|
| 期刊密度多面板 | 7-9 |
| 大面板柱状图 | 24 |
| 紧凑子图 | 15-16 |
| 轴标签（大面板） | 32-54 |
| 柱内标注 | 32-36 |
| 图例文本（大面板） | 28-38 |
| 刻度标签 | 20-36 |

## 2) 轴与边框

- 只保留左+下边框（Nature标准）
- 无网格线（偶尔稀疏y轴引导视线）

## 3) 色彩

**语义规则**: 蓝=提出方法，绿=正向变体，红=基线，中性=参考/背景。

**统一家族规则**（NMI风格推荐）：
- 相关基线用同一冷色家族
- 方法变体用同一暖色家族（lilac/rose）
- 绿/红仅用于方向性注释（提升/下降）

详见 SKILL.md 中的色板定义。

## 4) 排版

**图尺寸指南**：
| 类型 | figsize (inch) |
|------|---------------|
| 期刊宽度复合页 | (7.0-7.4, 5.5-7.8) |
| 多指标柱状（3-4指标） | (28-45, 6-12) |
| 紧凑单柱 | (9-16, 5-8) |
| 趋势/折线多面板 | (14, 4) 或 (9, 8) |
| 热图 | (8-20, 5-9) |
| 雷达图 | (12, 10) |

## 5) 核心原则

- 每个面板回答独特的科学问题
- 面板按证据值排序大小（不等分）
- 图例放在专用面板或共享条带
- Y轴紧缩到数据范围（不是自动0-100）
- SVG优先格式（可编辑文本）
- 使用相同条件/方法的颜色跨面板一致
