# 小红书卡片生成模式 (Xiaohongshu Card Patterns)

## 适用场景

项目推广、技能展示、科研成果宣传——生成竖版多图轮播，用于小红书发帖。

## 技术规格

| 参数 | 值 | 说明 |
|:-----|:---|:------|
| 分辨率 | **1080×1920 px** | 9:16 竖版，小红书轮播标准 |
| 模式 | RGBA | 支持透明渐变叠加 |
| 背景色 | (13, 17, 28) | 深海军蓝 |
| 卡片底色 | (22, 28, 46) | 略浅的卡片背景 |
| 强调色 | (0, 188, 212) | 青色/teal |
| 强调暗色 | (0, 140, 160) | 青色暗版 |
| 正文字色 | (200, 210, 230) | 浅灰蓝 |
| 字体 | NotoSansCJK-Bold.ttc (≥30pt) | 中日韩+拉丁混合渲染 |

## 布局模板

```
┌────────────────────────┐
│ (40,60) ── 修饰角 ──   │  ← 左上角 L形装饰线
│                         │
│   ┌──────────────┐     │
│   │ 标签 Badge   │     │  ← 圆角标签，青色底
│   └──────────────┘     │
│                         │
│  ─── 强调线 ───         │  ← 短水平线
│  主标题 (48pt Bold)    │  ← 粗体居中
│  副标题 (30pt Light)   │  ← 青色副标题
│                         │
│  ┌─────────────────┐   │
│  │                 │   │
│  │  正文内容区域    │   │  ← 圆角卡片 (r=24)
│  │  圆角背景        │   │
│  │                 │   │
│  │  # 小节标题     │   │  ← 白色粗体+短强调线
│  │  !! 高亮行       │   │  ← 青色粗体（双描）
│  │  > 次要信息      │   │  ← 灰色细体
│  │  普通行          │   │  ← 浅灰蓝常规体
│  │                 │   │
│  └─────────────────┘   │
│                         │
│  ─────────────────────  │  ← 底部色带
│ 页码: 1/5        尾角 ──│
└────────────────────────┘
```

## 文本格式规则

| 前缀 | 含义 | 渲染效果 |
|:-----|:-----|:---------|
| `!!` | 高亮重点 | 青色粗体 + 双描 |
| `> ` | 次要/元信息 | 灰色细体，缩进感 |
| `# ` | 小节标题 | 白色粗体 + 短强调线 |
| 无前缀 | 普通正文 | 浅灰蓝常规体，居中 |

## 视觉元素

1. **背景径向光晕**: 从中心偏上 (cx=W/2, cy=H*0.2) 向外渐隐的青色半透明椭圆环
2. **微网格**: 60px 间距的极暗线条网格 (18,22,35)，增加科技感
3. **圆角卡片**: 正文区域 r=24 圆角矩形，深色底 + 内边距 50px
4. **修饰角**: 左上/右下 L形 3px 短线，减少留白空旷感
5. **强调线**: 标题前的短横线 (w=80, h=4, r=2)，分割视觉层次
6. **底部色带**: 页面底部 20px 青色条带，收束视觉流
7. **页码**: 右上角 "1/5" 样式，灰暗色

## 生成流程

```python
# 标准入口
def draw_card_base(name, subtitle, body_lines, tag="", card_num=1, total=5):
    img = Image.new("RGBA", (W, H), BG)   # 1080×1920
    draw = ImageDraw.Draw(img)
    draw_glow_bg(draw)                     # 径向光晕
    draw_grid(draw)                        # 微网格
    draw_corners(draw)                     # 修饰角
    draw_tag(draw, tag)                    # 标签badge
    draw_accent_line(draw, y)              # 强调线
    draw_title(draw, name, subtitle)       # 主副标题
    draw_body_card(draw, body_lines)       # 圆角卡片+正文
    draw_bottom_bar(draw)                  # 底部色带
    draw_page_number(draw, card_num, total) # 页码
    return path
```

## 注意事项

- **NotoSansCJK-Bold.ttc 在 ≥30pt 渲染效果最佳** — 混合中英文时两边都清晰
- **双描(double-draw)** 解决CJK字体在暗色背景上笔画偏淡的问题：`draw.text((x,y)); draw.text((x+1,y))`
- **避免使用 emoji** — Linux 系统字体把 emoji 渲染成窄豆腐块
- **色彩范围验证**：生成后用 `img.getextrema()` 确认全通道有0-255动态范围，证明内容不是纯黑
- **亮度像素计数**：截取文字区域，统计 RGB >150 的像素比例，典型值 2-3%（在大面积深色背景下合理）
