# NSFC Grant Review — 8-Dimension Scoring Rubric

## 1. 立项依据 (7/10 baseline)

| Score | Criteria |
|-------|----------|
| 9-10 | Clear scientific problem; deep literature review; strong logical chain from clinical need → mechanism; well-cited, diverse sources |
| 7-8  | Good foundation but missing one link in logical chain; literature is adequate but not systematic |
| 5-6  | Scientific problem vague; literature review is superficial; no clear gap identified |
| 1-4  | No coherent scientific problem; literature barely cited; no logical argumentation |

## 2. 研究内容 (7.5/10 baseline)

| Score | Criteria |
|-------|----------|
| 9-10 | Content fully aligned with scientific problem; includes sub-type specificity; clear methodology for each content item |
| 7-8  | Good alignment but missing sub-type or methodological detail |
| 5-6  | Partial alignment; some content feels like an afterthought |
| 1-4  | Disconnect between problem and content; content is generic |

## 3. 技术路线 (7/10 baseline)

| Score | Criteria |
|-------|----------|
| 9-10 | Clear path; addresses overfitting/dimensionality; multi-modal fusion strategy explicit; validation strategy rigorous |
| 7-8  | Reasonable but missing detail on fusion strategy or validation |
| 5-6  | Too vague; no plan for overfitting; fusion strategy not mentioned |
| 1-4  | Technically infeasible or poorly described |

## 4. 创新性 (7.5/10 baseline)

| Score | Criteria |
|-------|----------|
| 9-10 | Innovation positioned as "pathological mechanism innovation"; novel biomarkers/methods; clearly differentiated from existing work |
| 7-8  | Innovation present but framed as "engineering improvement" rather than scientific advance |
| 5-6  | Incremental; no clear differentiator |
| 1-4  | No real innovation; just applying known methods to a new disease |

## 5. 研究基础 (7/10 baseline)

| Score | Criteria |
|-------|----------|
| 9-10 | Prior work directly relevant; clear linkage between past publications/projects and current proposal |
| 7-8  | Relevant experience but linkage not explicitly stated |
| 5-6  | Some experience but weak connection to proposed work |
| 1-4  | No relevant prior work demonstrated |

## 6. 预期成果 (4/10 — frequent weak point)

| Score | Criteria |
|-------|----------|
| 9-10 | Multi-dimensional: dataset + technology/IP + academic output; specific, quantifiable targets |
| 7-8  | Mostly complete but missing one dimension (e.g., no IP) |
| 5-6  | Only academic output; no dataset or technology transfer |
| 1-4  | Vague; non-quantifiable; or just "1-2 papers" |

## 7. 预算与进度 (3/10 — critical failure point)

| Score | Criteria |
|-------|----------|
| 9-10 | Budget matches NSFC standards (青年~30万, 面上~50万); structured as 设备费/业务费/劳务费; timeline realistic |
| 7-8  | Budget reasonable but structure slightly off; timeline OK |
| 5-6  | Budget too low; wrong structure; timeline unrealistic |
| 1-4  | Budget extremely low (e.g., 1万); completely wrong format |

## 8. 其他 (bonus)

- Risk assessment and mitigation plan present
- Ethics considerations for human subjects research
- Visual aids (flowcharts, diagrams, tables)
- Reference formatting (GB/T 7714-2015)
- Overall readability and professional tone
