# 申报书双质量检查协议（7维适配版）

> 2026-05-25实战验证：RFID医院资产管理标书，初始校准分0.30→修复后0.90
> 从sci-paper-quality-review的7维评审适配为申报书评分体系

## 与论文双质检的区别

| 维度 | 论文版 | 申报书版 |
|:-----|:-------|:---------|
| D1 | 科学贡献 | **科学问题/研究空白真实性** |
| D2 | 方法学严谨性 | **方法学严谨性（含预算可行性）** |
| D3 | 结果可信度 | **数据可信度（L0.5声明）** |
| D4 | 完整性 | **格式完整性（对标原模板）** |
| D5 | 清晰性 | **逻辑清晰性** |
| D6 | 新颖性 | **创新性与预算匹配度** |
| D7 | 引用质量 | **参考文献质量（PMID/DOI可验证）** |

## 校准时注意

- 市局级（1-5万）：校准分≥0.70 = PASS
- 省科技厅（10-30万）：校准分≥0.75 = PASS
- NSFC（30-50万+）：校准分≥0.80 = PASS

## 关键陷阱

### D2/D3：UHF RFID定位准确率声称

常见错误：声称UHF RFID可实现≥99%空间定位准确率。
事实：UHF RFID是区域级门禁盘点技术，不是连续空间定位技术。
正确写法：门禁盘点准确率≥95%（区域级），核心指标放在API数据同步一致率。

### D4：格式对标铁律

必须严格对标原始模板的七节格式（一立项、二现状、三内容、四目标、五方案、六进度、七基础），不得改为空白→假设→原子分解→门控等Synthos格式。Synthos理念通过内容嵌入实现（假设嵌入研究方案、L0.5声明嵌入预期目标、门控嵌入进度安排）。

### D7：参考文献验证

- 每篇必须标注PMID或DOI
- 用PubMed E-utilities做年份±1容错搜索
- 20条覆盖6个方向为合格
- <10条或无法验证的申请立即降级
