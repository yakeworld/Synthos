# NSFC Budget Templates by Category

## 青年科学基金项目 (Youth Fund)
**Typical total: 20-30 万元 (direct costs)**

| 科目 | 建议范围 | 典型用途 |
|------|----------|----------|
| 设备费 | 3-8万 | 专用设备购置/租赁、传感器、数据采集设备 |
| 业务费-临床测试与耗材 | 2-5万 | 量表耗材、受试者补贴、检测试剂 |
| 业务费-出版/文献/知识产权 | 1-3万 | 论文发表、专利申请、文献检索 |
| 业务费-数据采集与存储 | 1-2万 | 云端存储、GPU算力、数据标注 |
| 业务费-差旅/会议 | 1-2万 | 学术交流、会议报告 |
| 劳务费 | 8-12万 | 学生劳务、科研助理、数据标注人员 |
| **合计** | **20-30万** | |

## 关键数值参考

| 受试者规模 | 人均成本估算 | 说明 |
|------------|-------------|------|
| 50例 | ~0.3-0.5万 | 含基础量表+简单检测 |
| 150例 | ~0.5-0.8万 | 含多模态数据采集 |
| 300例 | ~0.8-1.2万 | 含随访、多模态、复杂分析 |

## 常见预算错误

1. **总预算过低** (<10万): 对任何涉及临床受试者的项目都不现实
2. **劳务费不足**: 300例×2年的数据采集+标注+随访需要充足人员投入
3. **设备费缺失**: 高精度眼动仪、传感器等往往需要专门购置
4. **业务费结构不清**: 应按NSFC标准科目分类，不要用自由格式
