# SCC Mathematical Morphology — 实战记录 2026-05-28

## 研究问题
半规管空间形态符合什么数学模型？从中心线数据→数学公式→流体动力学理论。

## 数据
GitHub: yakeworld/inner_ear_canal
- micro-CT中心线（3个半规管 × 骨性/膜性 = 6条管道）
- 3D Slicer导出.mrk.json格式，LPS坐标系

## 分析方法

| 分析 | 方法 |
|:-----|:------|
| 平面度 | SVD最佳拟合平面，RMS偏差/弧长比 |
| 形状拟合 | 投影到2D→圆vs椭圆拟合，AIC/BIC比较 |
| 曲率/挠率 | Frenet-Serret标架（中心差分） |
| 模型比较 | 6种模型：圆/椭圆/非平面椭圆+正弦/螺旋/Fourier |

## 最优数学模型

**Level 2: 非平面椭圆弧 + 正弦扭转（10参数，RMSE 0.08-0.16 mm）**

```
r(t) = R · (a·cos(t), b·sin(t), A·sin(ωt+φ))ᵀ + c
```

| 参数 | AC | PC | LC |
|:-----|:--:|:--:|:--:|
| a, b (mm) | 2.2, 3.6 | 3.6, 3.1 | 3.0, 2.5 |
| A (mm) | ~0.2 | ~0.2 | ~0.15 |
| ε=a/b | **1.63** | 0.88 | 0.84 |

## 关键发现
1. 椭圆拟合改善63-74%（vs圆）
2. 非平面度1.2-1.7%（轻度但可测）
3. 膜性管挠率显著高于骨性（1.8-2.6 vs 0.14-0.33 mm⁻¹）
4. Frenet挠率非零→三维扭转存在

## 流体动力学理论

[完整修订版理论见 NotebookLM项目 d9b4d992]

**核心命题**: 椭圆截面在Stokes流（Re<<1）条件下解耦灵敏度-阻尼。

椭圆管Poiseuille流:
- 灵敏度S ∝ A = π·ε·b² （随ε↑增加）
- 阻尼C ∝ (1+ε²)/(2ε) （随ε↑增加）
- 对比圆管：缩小r→S下降, C增加→无法兼得
- 椭圆策略→S和C同时增大→带宽扩展

**非平面扭转效应**: 抑制经典扭转摆在~10Hz的共振峰。
