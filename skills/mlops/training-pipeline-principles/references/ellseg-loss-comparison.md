## Loss Composition (EllSeg's `get_allLoss`)

EllSeg uses a **4-component composite loss**:

```
loss = l_seg + 0.1·l_seg2pt + 0.01·l_pt + l_seg2el
```

### 1. Segmentation Loss: `α·SurfaceLoss + (1-α)·GDice + wCE`

| Component | Function | Edge-sensitive? |
|-----------|----------|:---------------:|
| **SurfaceLoss** | Multiplies softmax output by **signed distance transform** of GT boundary. Pixels far from boundary get larger penalty. | ✅✅✅ |
| **GDice** (Generalized Dice) | Standard Dice over all classes. | ❌ |
| **wCE** (weighted Cross Entropy) | CE with spatial weights (boundary-enhanced). | 🟡 |

**SurfaceLoss formula:**
```python
def SurfaceLoss(x, distmap):
    x = torch.softmax(x, dim=1)           # [N, C, H, W]
    score = x * distmap                    # multiply by distance transform
    score = torch.mean(score, dim=2)       # mean over pixels per channel
    score = torch.mean(score, dim=1)       # mean over channels
    return score
```

The distance map is generated by `one_hot2dist`:
```python
def one_hot2dist(posmask):
    negmask = ~posmask
    res = distance(negmask) * negmask - (distance(posmask) - 1) * posmask
    return res / maxDist  # normalize by image diagonal
```

### 2. Seg-to-Point Loss (L1 on Center of Mass)
```python
l_seg2pt_pup = L1(soft_argmax(op[:,2]), pupil_center)  # pupil center from seg
l_seg2pt_iri = L1(soft_argmax(iriMap), iris_center)    # iris center from seg
```
Uses soft-argmax (temperature=4) to extract differentiable center from segmentation.

### 3. Point Loss (L1 on Ellipse Parameters)
```python
l_pt = L1(elOut, elNorm)  # direct L1 on all 10 ellipse params
```
Low weight (0.01×) — acts as mild regularizer.

### 4. Seg-to-Ellipse Consistency (BCE)
```python
# Render ellipse as soft mask, compare with segmentation via BCE
# Dual: posmask (outside ellipse) + negmask (inside ellipse)
loss += BCE(posmask, 1-seg) + BCE(negmask, seg)
```
This is exactly what our `param_seg_dice` does — but using **BCE** not Dice. 
The BCE approach gives **per-pixel independent gradients** — edge pixels contribute proportionally, not drowned by interior.

### Alpha Curriculum

```python
alpha = linVal(epoch, (0, args.epochs), (0, 1), 0)
# alpha=0 at epoch 0 → alpha=1 at final epoch
# Starts at pure GDice (fast convergence) → ends at pure SurfaceLoss (edge refinement)
```
