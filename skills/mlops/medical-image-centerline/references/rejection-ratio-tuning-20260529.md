# Rejection Ratio Tuning on HBL uCT (2026-05-29)

## 背景

Human_Bony_Labyrinth 数据集包含22个标本(F01-F14 + T01-T08)，每个标本同时有CT和uCT两种模态的标签。
体素间距：uCT=0.061mm, CT=0.2-0.5mm。

本实验对v5管线 `extract_centerlines()` 的 `ampulla_rejection_ratio` 参数做系统调优。

## 参数语义反转

**关键理解**：`_reject_ampulla` 实现是语义反转的。

```python
n_reject = int(n * ratio)               # e.g., n=100, ratio=0.4 → n_reject=40
threshold = np.sort(radii)[n_reject]    # 40th smallest radius value
keep = radii <= threshold               # keep ~40% narrowest nodes
```

`ratio` 实际控制**保留比例**（keep_ratio）而非剔除比例。ratio越高，保留的节点越多，弧越长。
直接后果：ratio从0.35升到0.50，弧长从~6mm增至~10mm。

## 调参协议

1. **先单例扫全范围**：选F01，跑 ratio=[0.25, 0.30, 0.35, 0.40, 0.45, 0.50, 0.55]
2. **检查AC/PC分离度**：centroid距离<1mm = MERGE
3. **在多个case验证窄范围**：F01/F05/F08/T01/T04，ratio=[0.35, 0.40, 0.45]
4. **同时跑CT对照**：同case的CT在相同ratio下的弧长作为参考

## 结论

- **HBL uCT 最优值: rejection_ratio=0.40**
  - 弧长：AC~7.5, PC~7.2, LC~6.9mm (CT的~70-90%)
  - AC/PC分离度：5/6 case >4mm
- **ratio≥0.45 不可用**：AC/PC共用common crus → PCA命名法输出相同坐标
- **F02为解剖特例**：所有ratio下AC/PC均无法分离（可能三管高度共路）
- **F05的LC偏短**(~1.7mm)：可能是提取角度问题，非ratio所致
