# 学术论文质量评审与格式审查

## 文言

| 本体 | 文言 | 义 |
|:-----|:-----|:----|
| 逐页提取 | **逐页识文，不遗一字** | PDF全量提取后分段标注页码 |
| 质量与格式齐审 | **器质并查，不偏不废** | 同时检查方法学可靠性和呈现规范性 |
| 优先排序 | **危先缓后，不骛虚务** | P0/P1/P2分级输出修复建议 |

## 触发条件

用户发送PDF文件（尤其是中文医学硕士/博士学位论文）且要求"质量评审"、"格式审查"、"审稿"、"检查"时加载。

文件通过Feishu(飞书)发送到 `~/.hermes/cache/documents/` 目录。

## 工作流

### Phase 1: PDF全文提取

```bash
cd ~/.hermes/cache/documents
python3 -c "
import fitz
doc = fitz.open('FILENAME.pdf')
print(f'Total pages: {len(doc)}')
for i in range(len(doc)):
    text = doc[i].get_text()
    if text.strip():
        print(f'===== PAGE {i+1} =====')
        print(text[:2000])
"
```

### Phase 2: 结构完整性检查

| 检查项 | 方法 |
|:-------|:-----|
| 目录与实际内容对应 | 目录页列出的章节≥90%在正文中找到 |
| 中文摘要四段式 | 目的/方法/结果/结论齐 |
| 英文摘要对应 | 与中文摘要信息一致 |
| IMRaD结构 | 引言→材料/方法→结果→讨论→结论 |

### Phase 3: 格式审查（8项必查）

1. **参考文献编号冲突** 🔴 P0
   - 中文医学论文常见问题：正文参考文献[1]-[N]与末尾综述的参考文献[1]-[M]编号重叠
   - **检查方法**：用Python正则 `re.findall(r'\[(\d+)\]', main_text)` 提取所有引用编号，与参考文献列表编号比对
   - **修复**：综述部分改用独立编号体系（罗马数字I-LXVI或字母A1-A66）

2. **知情同意缺失** 🔴 P0
   - 伦理委员会批准号有，但需额外注明患者知情同意
   - 回顾性研究也应注明"经伦理委员会批准豁免知情同意"

3. **英文标题/摘要不一致** 🟡 P1
   - 中文标题含"良性"则英文也应含Benign
   - 英文摘要缺少临床具体数据（成功率/例数等）

4. **表/图编号完整性** 🟡 P1
   - `re.findall(r'表\d+', text)` 和 `re.findall(r'图\d+', text)` 提取
   - 检查是否有跳号（如表1→表3缺表2）
   - 检查所有编号在正文中是否至少出现一次

5. **缩略词表完整性** 🟡 P1
   - 正文首次使用缩略词时应有全称
   - 缩略词表应覆盖正文使用≥3次的缩略词

6. **页码/页眉一致性** 🟢 P2
   - 每页页眉应为同一内容
   - 页码从正文第一页开始连续

7. **章节编号层级** 🟢 P2
   - 一级用一/二/三…，二级用1.1/1.2…，三级用1.1.1/1.1.2…
   - 缺层（如从一级跳到三级无二级）需标记

8. **参考文献格式** 🟢 P2
   - 检查是否遵循GB/T 7714或目标期刊格式
   - 期刊名缩写一致性

### Phase 4: 质量评估（7维）

| 维度 | 评分 | 检查要点 |
|:-----|:----:|:---------|
| 创新性 | 1-5⭐ | 新概念/新方法/新发现 |
| 方法学严谨性 | 1-5⭐ | 样本量/对照/盲法/随机化 |
| 统计分析 | 1-5⭐ | 有假设检验(p值/置信区间)还是纯描述 |
| 结果呈现 | 1-5⭐ | 表图质量/数据完整性 |
| 讨论深度 | 1-5⭐ | 与文献对比/机制解释/局限 |
| 文献覆盖 | 1-5⭐ | 数量/时效/相关性 |
| 临床价值 | 1-5⭐ | 转化潜力/实践意义 |

### Phase 5: 优先级修复建议

```
P0 (答辩前必改):
  - 参考文献编号冲突 → 综述编号改体系
  - 知情同意缺失 → 补充豁免说明
  - 英文标题不一致 → 补全对应

P1 (建议修):
  - 缺乏统计推断 → 加假设检验
  - 英文摘要缺数据 → 补临床数字
  - 某页内容提取不全 → 确认PDF渲染

P2 (可优化):
  - 页码格式统一
  - 参考文献格式微调
```

## 实测陷阱

1. **PDF页面内容提取不全**：包含大量图表/公式的页面（如中文摘要续页、致谢页）可能只提取到几个字。用pdftotext二次提取或用`doc[i].get_images()`检查是否是纯图片页面。
2. **中文医学论文引用编号重叠**：正文参考文献[1]-[N]与末尾综述文献[1]-[M]是两套体系。这在格式审查中是**最高优先级**问题，因为答辩委员/外审专家无法区分。
3. **排除标准和纳入标准**：检查正文时别只搜中文——有些论文英文术语（inclusion/exclusion criteria）混用。
4. **样本量不一定是硬伤**：但必须确认有样本量估算或明确说明样本来源。
5. **综述末尾的参考文献格式**：有时GB/T 7714标准中的期刊名缩写不一致（如JAMA vs Journal of the American Medical Association）。

## 常见问题速查（2026-06-03实战补充）

### 虚拟仿真/生物力学论文的评审要点

当论文使用三维虚拟仿真（Unity 3D/PhysX/有限元）时：

| 检查项 | 要点 |
|:-------|:-----|
| 模型来源 | micro-CT / MRI / 公开模型？分辨率多少？ |
| 参数设定 | 耳石密度/黏度/尺寸是否有文献依据？单颗粒vs多颗粒？ |
| 验证方式 | 仿真结果是否有独立临床数据验证？ |
| 模型局限 | 是否讨论理想化假设（牛顿流体、球形耳石等）的影响？ |
| 四元数/旋转矩阵使用 | 头位转换是否基于实测体位角度的数学建模？ |

### 临床验证部分的统计方法评估

常见医学论文只有描述性统计（均数±标准差）而无假设检验：

| 信号 | 判定 | 建议 |
|:-----|:-----|:------|
| 组间比较仅%无p值 | ⚠️ 统计不完整 | 加Mann-Whitney U或t检验 |
| 只有总成功率的"n/N" | ⚠️ 缺乏精确性 | 加95%置信区间 |
| 亚组样本量<30 | ⚠️ 统计功效不足 | 需样本量估算或讨论局限 |
| 单中心回顾性+无对照组 | ⚠️ 设计偏倚 | 需诚实地在局限段讨论选择性偏倚 |

### 概念定义清晰性检查

当论文构建新概念（如"零点平面"、"阶梯DH试验"）时：

- ✅ 新概念是否在首次出现时给出明确定义？
- ✅ 定义是否基于可量化的几何/物理参数？
- ✅ 新概念在后续论证中被一致使用（而非一次过就抛弃）？
- ⚠️ 新概念是否与现有文献中的类似概念有明确区分？
- ❌ 概念听起来新颖但缺乏定量基础 → 建议给出数学表达式或图例说明

以下问题应在答辩前强制修复，用速查表替代逐段分析：

