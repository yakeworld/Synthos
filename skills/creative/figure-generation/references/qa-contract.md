# QA审核清单

最终交付前、修订包前、含显微/电泳/临床亚组/统计声明时使用。

## 提交前检查

| 检查项 | 通过条件 |
|--------|---------|
| 结论Claim | 一句可证伪主张存在，每个面板映射到唯一证据 |
| 原型声明 | 图表有声明的原型和面板层级 |
| 最终尺寸 | 单栏~89mm 或 双栏~183mm |
| 文本尺寸 | 最终尺寸下5-7pt可读（期刊密度） |
| 面板标签 | 小写加粗，左上角，最终尺寸~8pt |
| 可编辑文本 | SVG/PDF文本保持可编辑 |
| 字体 | Arial/Helvetica/sans-serif一致 |
| 色彩 | 无彩虹色板；红绿不是唯一编码；灰度打印可理解 |
| SVG/PDF | 至少主格式可编辑矢量 |
| 图像完整性 | 全局调整；无局部增强/遮盖 |
| 统计 | 样本量、误差棒定义、检验方法已注明 |

## 常见失败模式

| 模式 | 表现 | 修复 |
|------|------|------|
| 冗余面板 | 面板b只是面板a的不同形式 | 用Z-score热图替代绝对值展示 |
| 图例过载 | 图例含12+条目，每个面板重复 | 用直接标签或共享条带图例 |
| 色彩混乱 | 每个方法不同色相 | 用统一家族色（如NMI淡色系） |
| 不等可比 | AUC范围0.85-0.95但Y轴从0开始 | 紧缩到数据范围 |
| 审美驱动 | 先选模板再适配数据 | 先结论再选原型 |
