# Review Paper — Three-Pronged Hypothesis Template

> 综述论文（Critical Review / Methodological Framework）的科学假设结构与原创研究不同。不是"If X then Y"的实验假设，而是**三个相互竞争的论证立场（Thesis Positions）**，通过文献证据进行淘汰。

## 结构模板

### H₁：主推进假说（Advocacy）

**提出一个新方法论/维度/框架的价值。**

> **If** `[新维度/方法]` 被系统性纳入 `[领域]` 的评估体系，**then** 可以识别出传统 `[已有方法]` 未能捕捉的 `[新特征]`，**which** 对 `[应用场景]` 的诊断/预测价值不低于 `[已有方法]`。

**可证伪条件**：若已有文献中有 ≥N 项研究使用了该方法却发现无显著差异（p>0.05 / AUC < 阈值），则 H₁ 被证伪。

### H₂：冗余假说（Redundancy）

**新维度与已有维度存在高度信息重叠，不提供独立增量价值。**

> `[新维度]` 与 `[已有维度]` 高度相关（r > 0.7），本质上是同一神经/功能环路的表征冗余。额外测量不提供独立于现有参数的信息。

**可证伪条件**：若至少一篇研究证实 `[新维度]` 的诊断AUC ≥ 0.80 且与所有传统参数的相关系数 r < 0.5，则 H₂ 被证伪。

### H₃：噪声假说（Noise）

**新维度的信号-噪声比过低，不足以支撑个体水平判定。**

> `[新维度]` 的幅度接近当前测量设备的噪声底限，其个体间变异小于测量误差，无法在个体水平上提供可靠的临床判定。

**可证伪条件**：若已有研究在个体水平上使用该维度实现了分类准确率 ≥ 80%，则 H₃ 被证伪。

## Stress Test 设计

构建一个**证据层级表**，将多个待比较的参数/维度按以下标准评级：

| 指标 | 权重 | 评分标准 |
|:-----|:----:|:---------|
| 效应量 | 0.30 | Cohen's d ≥ 0.8 = 3分, 0.5-0.8 = 2分, <0.5 = 1分 |
| 可复制性 | 0.25 | 独立验证 ≥ 3次 = 3分, 1-2次 = 2分, 仅1篇 = 1分 |
| 神经解剖学对齐 | 0.25 | 与病理区域直接重叠 = 3分, 间接相关 = 2分, 无明确关联 = 1分 |
| 设备可及性 | 0.20 | 商用低成本 = 3分, 科研级 = 2分, 原型机 = 1分 |

**判定逻辑**：
- H₁ 优势：新维度在 ≥ 3 项指标上优于任意传统维度
- H₂ 优势：新维度与某传统维度的相关系数 r > 0.7 且 AUC 无显著差异
- H₃ 优势：新维度的个体内变异 / 个体间变异 > 1.0

## 使用场景

| 论文类型 | 适用性 | 示例 |
|:---------|:------:|:-----|
| Critical Review | ✅ 首选 | "三维扭转作为PD眼动标志物的缺失维度" |
| Methodological Framework | ✅ 首选 | "三维眼动追踪方法论框架" |
| Systematic Review | ⚠️ 可适配 | 将H₁改为"已有证据支持该维度"，H₂改为"证据不足" |
| Narrative Review | ⚠️ 可适配 | 建议升级为Critical Review以增强论证力度 |

## 示例：PD眼动标志物综述

- **H₁**（主假设）：三维扭转参数作为PD早期筛查标志物具有未被开发的临床价值，其敏感性不低于传统二维扫视参数
- **H₂**（冗余假说）：扭转维度与传统二维眼动参数高度相关，本质上是同一神经环路损伤的表征冗余
- **H₃**（噪声假说）：扭转幅度（<3°）接近商用3D眼动仪噪声底限（0.1-0.2°），信号-噪声比过低
- **Stress Test**：构建证据层级表，比较扫视潜伏期、SWJ、OMT、扭转幅度等6-8个参数在效应量/可复制性/解剖对齐/设备成本四维上的总分
