# 政策对齐工作流程（Policy Alignment Workflow）

## 目的

将用户（临床医生/科研者）的技术与临床资产，与政府产业规划方案/政策文件中的任务板块进行逐条映射，找到"嵌入点"，确定发言策略和申报定位。

## 适用场景

- 政府产业规划座谈会（如脑机接口、人工智能、生物医药等）
- 政策方案初稿讨论会
- 专项资金/人才计划申报前的策略准备
- 政策方征求行业意见时的结构化反馈

## 工作流程

### Phase 1: 政策方案解析

1. 读取方案全文，结构化提取：
   - **战略定位**：政策方希望达成什么目标？
   - **主要任务**：每个子项（如：平台引领、产业孵化、临床资源、公共服务、材料研发）
   - **牵头单位**：每个任务板块的主导方是谁？
   - **保障措施**：资金、人才、审批、产学研合作等
   - **议事协调机构**：组织架构中的位置

2. 标注每个任务板块的"关键词"：
   - 疾病方向（如：帕金森、阿尔茨海默、睡眠障碍、抑郁）
   - 技术方向（如：脑机接口、眼动追踪、神经调控、柔性电极）
   - 临床场景（如：临床试验、康复病房、患者队列）
   - 产业场景（如：孵化器、企业培育、材料制造）

### Phase 2: 用户资产盘点

读取用户的以下材料：
- 技术/硬件：专利、原型机、设备规格
- 临床能力：患者数据、临床验证、诊疗设备
- 论文产出：SCI论文清单（按方向分类）
- 平台建设：实验室、研究中心、联合实验室
- 团队：人数、学科背景、人才梯队
- 科研方法论：AI驱动科研体系（如Synthos）
- 产业化：已转化产品、销售收入、量产能力
- 人才需求：在研/计划中的博士/博士后需求

### Phase 3: 逐条映射

为每个政策任务板块创建匹配表：

| 政策任务 | 用户匹配度 | 可承担板块 | 发言要点 | 已有闭环 |
|:---------|:----------:|:----------|:---------|:---------|
| XX任务   | ⭐⭐⭐⭐⭐   | 具体板块名称 | 核心论点 | 是/否 |

匹配度评级：
- ⭐⭐⭐⭐⭐ 直接覆盖，已有完整成果
- ⭐⭐⭐⭐ 部分覆盖，需补充1-2项
- ⭐⭐⭐ 间接相关，可跨领域连接
- ⭐⭐ 有交集但薄弱，需大量补充
- ⭐ 基本不相关

### Phase 4: 识别机会

1. **空白机会**：政策中提到但用户尚未覆盖的方向 → 未来申报/论文选题
2. **已有闭环**：用户已完成临床转化的方向 → 强调"可复制性"，争取牵头
3. **技术壁垒**：用户独有的硬件/算法/专利 → 作为核心竞争点
4. **人才匹配**：用户的人才需求与政策"引育"章节的对齐

### Phase 5: 输出策略

1. **竞争力分析表**：用户在各评价维度的匹配度
2. **建议承担板块**：具体可争取的政策任务
3. **讨论会提问清单**：3-5个关键问题（涉及牵头单位、参与角色、资源分配）
4. **3-5分钟发言草稿**：开场白 + 核心论点 + 建议板块 + 结语

## 关键原则

1. **"已有闭环" > "未来规划"**：政府方案更看重已落地的成果，而非愿景
2. **"技术壁垒" > "单一论文"**：打包专利+设备+临床数据+论文，形成完整技术栈
3. **"联合牵头" > "被动参与"**：争取联合单位或核心成员身份，而非仅作为参与者
4. **"可复制性"是杀手锏**：用户已完成一个疾病的闭环，强调该路径可复制到政策中的其他疾病方向
5. **人才需求与政策挂钩**：用户在研/计划中的人才需求，正是政策"保障措施"中的人才引育部分

## 错误模式（Pitfalls）

1. **过度承诺**：不要声称政策中用户尚未开展的方向"已有基础"
2. **忽略牵头单位**：方案中已指定牵头单位（如温医大），技术方应定位为"核心成员"而非"主导"
3. **论文堆砌**：政策方案评审看重"技术+临床+产业化"组合拳，不是单纯论文数量
4. **未对标政策语言**：发言应使用政策方案中的术语（如"一核四基地"、"感知运动康复"、"临床转化"）
5. **忽略会议背景**：座谈会不是答辩会，目标是"建立存在感+找到嵌入点"，不是当场拿项目
6. **暴露过多临床数据**：不要写"年5000+门诊，3000+检查"等具体数字。改为"已建立从临床到机制到原型的完整闭环管线"。不列具体数据集名称。
7. **混淆手稿与已发表论文**：统一表述为"已完成手稿"/"已编译"，不写"已发表论文"。手稿是管线产物，不是交付物。
8. **文件路径含特殊字符**：中文文件名（引号、括号）在 shell 中必须用单引号包裹完整路径，否则 cp/pdftotext 报错
