## 色板

| 名称 | 色值 | 用途 |
|------|------|------|
| `blue_main` | `#0F4D92` | 主证据 / 核心数据 |

| `green_1` | `#DDF3DE` | 正向结果（淡） |
| `green_2` | `#AADCA9` | 正向结果（中） |
| `green_3` | `#8BCF8B` | 正向结果（强） |
| `red_strong` | `#B64342` | 基线 / 风险 / 误差 |
| `red_light` | `#F6CFCB` | 基线（淡） |
| `red_secondary` | `#E9A6A1` | 基线（次） |
| `gray` | `#767676` | 中性 / 参考 |

| `cyan` | `#42949E` | 第三方 / 补充 |
| `purple` | `#7B5EA7` | 创新 / 核心贡献 |
| `orange` | `#E8954A` | 工程 / 执行 |

### 领域专用色板

| 领域 | 基色 | 正向 | 风险 | 中性 |

### 代码用法

```python
# 色板常量（从 api.md 导入）
PALETTE = {
    "blue_main":      "#0F4D92",
    "blue_secondary": "#3775BA",
    "green_1": "#DDF3DE", "green_2": "#AADCA9", "green_3": "#8BCF8B",
    "red_1":   "#F6CFCB", "red_2":   "#E9A6A1", "red_strong": "#B64342",
    "gray":    "#767676", "gold":     "#FFD700", "cyan":    "#42949E",
    "purple":  "#7B5EA7", "orange":   "#E8954A",
}

# 从色值取 RGB
from matplotlib.colors import to_rgb
rgb = to_rgb(PALETTE['blue_main'])  # (0.0588, 0.3020, 0.5725)

# Alpha 变体
rgba = rgb + (0.3,)  # Semi-transparent
```

## 字体

### 标准字体栈（sans-serif 优先）

```
Arial → Helvetica → DejaVu Sans → sans-serif
```

### 字号标准（单栏图 89mm）

| 元素 | 字号 | 说明 |
|------|------|------|
| Figure 标题 | 9pt | 可选，通常用 legend 代替 |
| 子图标题 (a/b/c) | 8pt | 小写粗体，左上角 |
| 轴标题 | 7pt | 常规 |
| 轴刻度标签 | 6pt | 常规 |
| 图例文本 | 6pt | 常规 |
| 主文本（框内） | 7-7.5pt | 常规/粗体 |
| 次要文本 | 6.5pt | 说明文字 |
| 版本戳 | 5pt | 斜体，底部 |
| 统计标注 (p-value) | 6-7pt | 粗体 |

### 粗体规则

- **框标题** → 粗体
- **面板标签 (a/b/c)** → 粗体
- **统计显著性 (***p<0.001)** → 粗体
- **坐标轴标题** → 常规
- **坐标轴刻度** → 常规
- **图例文本** → 常规
- **次要说明文字** → 常规

## 线型

| 用途 | 线型 | 线宽 | 颜色 |
|------|------|------|------|
| 实线数据 | `-` | 1.5 | 语义色 |
| 虚线趋势 | `--` | 1.0 | 红基 `#B64342` 或灰 |
| 点划线 | `-.` | 0.8 | 灰 `#999999` |
| 网格线 | `-` | 0.3 | `#E0E0E0` |
| 边框 | `-` | 0.5-1.0 | `#555555` |
| 重点边框 | `-` | 1.5-2.0 | 语义色（如核心框紫色） |

## 标记符号

| 用途 | 标记 | 大小 | 颜色 |
|------|------|------|------|
| 数据点（默认） | `o` | 40-50 | 语义色，白边 |
| 异常值 | `D` (diamond) | 30 | 红基 `#B64342` |
| 趋势线顶点 | `s` (square) | 40 | 语义色 |
| 分组区分 | `o`, `s`, `^`, `D`, `v` | 30 | 语义色 |

## Hatch 模式（灰度安全）

| Hatch | 视觉效果 | 推荐用途 |
|-------|---------|---------|
| `/` | 斜线左上 | 第一组 |
| `\\` | 斜线右上 | 第二组 |
| `.` | 点 | 第三组 |
| `x` | 叉 | 第四组 |
| `o` | 环 | 第五组 |
| `+` | 十字 | 第六组 |
| `|` | 竖线 | 第七组 |
| `-` | 横线 | 第八组 |

## 使用方式

```python
# 导入
from style import PALETTE, plot_fonts, line_styles, markers, hatches

# 应用
plt.rcParams.update(plot_fonts())
bars = ax.bar(x, y, color=[PALETTE['blue_main'], PALETTE['green_3'], ...])
for i, bar in enumerate(bars):
    bar.set_hatch(hatches[i % len(hatches)])
```
