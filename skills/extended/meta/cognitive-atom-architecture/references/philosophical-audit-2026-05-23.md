## 审计一：哲学自洽性

### 发现的两个内部矛盾

| 矛盾 | 涉及原则 | 说明 | 方案 |
|:-----|:---------|:-----|:-----|
| **P3 vs 熵减律** | P3 (人机分层) + 熵减律 (主动推理） | P3要求人类确认不可逆动作；熵减律要求系统主动选择信息获取方向以降低不确定性。主动推理门被"阉割"为被动告警 | 在P3下建立"主动推断例外"机制——当不确定性超过阈值且提案无不可逆风险时，允许自动执行 |
| **P1 vs P2** | P1 (原子可复现) + P2 (演化上浮) | 吸收新技能会改变原子内部推理路径，破坏旧版本的"语义可复现"承诺 | 引入原子版本冻结协议——吸收后旧版本保留为frozen/，通过版本号路由 |

### 发现的一个盲区

**缺失本体论层 (Ontology Layer)** — 7+1框架本体论层全部是认识论（Epistemology: 如何思考），缺少本体论（Ontology: 领域包含什么客体与关系）。表现为：

- "VOR本质就是固视"这样的科学论断不知道归属哪里
- 宪法不可修改条款里混入了领域知识
- 系统无法区分"思维法则"和"领域事实"

**修复**: 将"VOR本质就是固视"等具体领域论断从宪法下放到应用层（NotebookLM 知识库 + skill references）。宪法只保留纯认识论规则。

### 发现的两个哲学-原子断裂

| 断裂 | 哲学宣告 | 原子真相 | 修复 |
|:-----|:---------|:---------|:-----|
| **VER 伪贝叶斯** | 宣称"贝叶斯置信度校准" | 实际是线性减法启发式 (先验0.70 - 反方0.15) | 替换为真实贝叶斯更新: 后验 = 先验 * 似然比 / 证据 |
| **HYP 证伪纸上谈兵** | 宣称"证伪主义" | 只生成文本上的证伪条件，缺乏执行沙盒 | 增加实验执行沙盒原子, 或将证伪条件形式化为可API验证的assertion |

## 审计三：协议可移植性

核心特征重新定义:

| 旧说法 (已废弃) | 新说法 |
|:---------------|:-------|
| Zero-Python: 纯SKILL.md，无Python代码 | **可移植格式**: 文言+SKILL.md，运行时自然生成Python脚本 |
| Agent是唯一运行时 | **格式即协议**: 任何读取Markdown的Agent都可执行 |
| 排除外部代码 | **任何Agent协议可吸收**: Claude Code的hooks、Codex的CLI模式、GitHub Copilot的slash commands |

**可移植性测试**: 另一个Agent（Claude Code / Codex / OpenCode）能否加载此SKILL.md并执行？
- 是 → 协议可移植
- 否（依赖Hermes特定工具）→ 需要适配，但格式本身始终可移植

### 协议翻译规则

| 外部模式 | 翻译为文言 | 翻译为SKILL.md |
|:---------|:----------|:---------------|
| Python DSL | 文言四字格言 | YAML frontmatter + 三层内容 |
| slash commands | 命令层英文命令 | 写入命令层(命令)·英文 |
| JSON config | 方法层白话表 | 写入方法层(方法)·白话 |
| hooks/events | 文言事件驱动 | allowed-tools + 事件条件 |
