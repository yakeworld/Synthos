# 6-Minute Demo Video Script Template

## Video Information
- Duration: 6 minutes
- Format: MP4, 720P+
- Resolution: 1920x1080
- Language: Chinese (with English subtitles recommended)
- Naming: 智能体名称.mp4

## Script Template

### [0:00-0:30] Opening and Problem Introduction
**Visual**: Black screen → fade in logo → title
**Narration**: "In [domain], [problem]. Current solutions are [pain points]. [Product Name] aims to solve this."

### [0:30-1:30] Architecture Overview
**Visual**: Architecture diagram animation (layer by layer)
**Narration**: "[Product] is based on [framework]. It consists of [X components]..."

### [1:30-3:00] Core Demo Part 1
**Visual**: Screen recording — input query → show results
**Narration**: "Let's see a demo. When we input [example]... The system simultaneously [action]... With traditional methods taking [time], [product] completes in [time]."

### [3:00-4:30] Core Demo Part 2
**Visual**: Visualization of key features
**Narration**: "The [feature] automatically [action]... For example, [concrete example]... This helps users [benefit]."

### [4:30-5:15] Unique Capabilities
**Visual**: Process animation
**Narration**: "What makes [product] unique is [unique feature]. Unlike [alternatives], [differentiator]."

### [5:15-5:45] Results and Impact
**Visual**: Data comparison charts
**Narration**: "In real-world testing: [stat 1], [stat 2], [stat 3]... Used by [user type] in [context]."

### [5:45-6:00] Closing
**Visual**: Logo + tagline
**Narration**: "[Product Name] — [tagline]. Thank you."

## Recording Tips
- Tool: OBS Studio for screen capture + editing software
- Background music: Light tech electronic music (volume 20%)
- Subtitles: Bilingual recommended
- Screenshots: Use anonymized/de-identified data
- Pacing: Keep narration at ~150 Chinese characters per minute
