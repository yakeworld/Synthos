# LLM生成假DOI实战案例 (pima-crispdm 2026-05-31)

## 背景

`paper-pipeline` 对 pima-crispdm 论文执行全流程后，生成的 `references.bib` 中有大量虚假DOI。

## 关键发现

45条引用中，15条(33%)是假DOI或错配DOI。

### 假DOI模式

| 类别 | 数量 | 示例 |
|:-----|:----:|:------|
| **完全虚构**（Crossrref查不到） | 11 | 10.1016/j.compbiomed.2025.109456, 10.1016/j.eswa.2024.123456 |
| **真DOI错配**（Crossrref查到但标题不符） | 2 | Tong2024 DOI指向胸片识别, Kalagotla2021 DOI指向药物安全 |
| **HTML伪装PDF** | 2 | Gr2024(Gr2024Random.pdf), Pranto2020 |

### 假DOI格式特征

```
10.1109/ACCESS.2023.3234567     ← 卷页=12345-12358（序列号）
10.1016/j.compbiomed.2025.109456  ← 卷号=109456（虚构）
10.1016/j.knosys.2025.112345     ← 卷号=112345（虚构）
10.1007/s12652-024-04567-8       ← 虚构论文
10.1177/14604582221098765        ← 虚构论文
```

## 三源修复成功率

| 源 | 成功率 |
|:---|:------:|
| SS搜索（Tier 1） | 4/15 (27%) |
| OpenAlex搜索（Tier 2） | 4/15 (27%) |
| PDF元数据提取（Tier 3） | 2/15 (13%) |
| 无法修复（需人工） | 5/15 (33%) |

## 最终结果

初始D9=35%，最终D9=98%（43/44）。

唯一无PDF保留的是Wirth2000（CRISP-DM经典方法论，无DOI无OA）。
