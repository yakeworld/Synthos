## 通用模式

评审 -> 找最低分维度 -> 只修一个维度 -> 重审

## 关键原则

1. 每轮只修复最低分维度 -- 不贪多
2. 修复后必须重审全部7维 (分数可能联动变化)
3. 最多3轮，不过降级目标
4. 修订前后数据记录到 review/ 目录
