# Synthos 7+1 东西合参认知框架 — 原始来源

来源: https://wiki.3deyes.top/index.php/Synthos (Revision 457, 2026-05-08)

> Synthos 是科学认知的数字生命体——一个由可执行技能构成、以契约为基因、以证伪为进化机制、以熵减为终极目标的自组织知识生态系统。

## 三大支柱

### 1. 技能即科学单元 (Skills as First Principles)
每项科研操作被封装为标准化的技能，遵循严格规范 (skill.yaml)。
- 机器可读 · 可测试 · 可复现 · 可审计
- 禁止黑箱 · 禁止无测试 · 禁止无版本

### 2. 智能体为契约执行者 (Agent as Contract Enforcer)
基于 Hermes Agent，Agent 不"生成"答案，而是匹配、组合、调用技能。
- 智能体不依赖常识来猜测，而以契约为第一准则
- 它能理解，但选择执行——因为唯有可验证的执行，才是科学的起点

### 3. 系统自我进化 (Self-Purification via Entropy Reduction)
- 每次技能调用，计算预测误差（输出是否符合预期？）
- 失败 → 降低信任度；成功 → 提升权重
- 每周对抗性测试；连续失败3次 → 自动归档
- 科学不是靠权威保存，而是靠错误淘汰

## 技术架构（简要）

| 层级 | 组件 | 说明 |
|------|------|------|
| 核心引擎 | Hermes Agent | 调度、推理、组合技能 |
| 技能库 | GitHub 组织 synthos/skills | 每个技能为独立仓库 |
| 执行环境 | Docker + CodeSandbox | 隔离运行 |
| 验证系统 | GitHub Actions | 自动测试、性能评估、信任评分 |
| 用户接口 | CLI / Web Demo | 自然语言输入 → 结构化输出 |
| 知识图谱 | Skill Network Map | 可视化技能依赖与共现关系 |
