# 原子依赖矩阵

## 严格 DAG 结构

```
     [1] → [2] → [3] → [4] → [5]
                    ↘      ↘
                     └──── [6]
```

## 依赖矩阵 (1=有依赖, 0=无依赖)

|     | 1  | 2  | 3  | 4  | 5  | 6  |
|-----|----|----|----|----|----|----|
| 1   | -  | 0  | 0  | 0  | 0  | 0  |  (根节点，无依赖)
| 2   | 1  | -  | 0  | 0  | 0  | 0  | (依赖 1)
| 3   | 0  | 1  | -  | 0  | 0  | 0  | (依赖 2)
| 4   | 0  | 0  | 1  | -  | 0  | 0  | (依赖 3)
| 5   | 1  | 0  | 0  | 1  | -  | 0  | (依赖 4 + 直接依赖 1)
| 6   | 0  | 0  | 0  | 1  | 1  | -  | (依赖 4 + 5)

## 拓扑排序

唯一拓扑排序: [1, 2, 3, 4, 5, 6] 或 [1, 2, 3, 4, 6, 5]

## 原子深度

| 原子 | 深度 | 说明 |
|------|------|------|
| 1 | 0 | 根节点 |
| 2 | 1 | 子节点 |
| 3 | 2 | 子节点 |
| 4 | 3 | 子节点 |
| 5 | 4 | 叶子节点 |
| 6 | 4 | 叶子节点 |

## 验证规则

1. **无循环**: 沿依赖方向不可能回到起点
2. **连通性**: 所有原子可达（从1出发能到达所有原子）
3. **叶子**: 5和6是最终输出，不依赖其他原子
4. **根**: 只有1是根（无依赖的原子）
5. **无跨层**: 依赖只能是当前层或上一层（允许直接依赖1到5）
