# 三维眼动分析研究生训练方案（加速版）

## 核心策略：以战代练

> 不花时间专门学工具，**用写论文驱动一切**。
> 工具不会 → OpenCode 生成 → 理解并修改 → 下一篇速度翻倍。

## 时间线（6-8周一篇文章）

```
第1周      第2周      第3周    第4周     第5周      第6周
选题确立 → 数据集预处理 → 分析管线跑通 → Method+Results → 全文初稿 → 修改投稿
           第一张图      完整的Table1   导师检查        一轮修改   完成
```

## 建议第一篇：三维 Kappa 角在正常人群中的分布特征

- 数据：公开数据集（GazeCapture / Lund2013）→ 不用等伦理
- 方法：已有 dual-ellipse 代码 → 批量处理 → 统计分析
- 医学背景优势：知道正常参考值临床意义

## 论文结构模板

```
Title: 3D Kappa Angle Distribution in a Large-Scale Public Eye-Tracking Dataset
Abstract: 150 words
Introduction: 500 words (临床需求 + 现有方法不足)
Methods: 数据描述 + 三维瞳孔定位 + Kappa角计算 + 统计
Results: 人群分布统计 + 年龄/性别差异 + 左右眼对称性
Discussion: 500 words (临床意义 + 局限性)
References: 20-30篇
```

## 注意事项

1. OpenCode 是工具，不是替代思考 — 每次产生结果后必须理解含义
2. 医学背景的优势 — 知道临床需要什么指标，这是做算法的人没有的
3. 所有实验代码和数据必须归档（按 Synthos 规范）
4. 第一篇不要追求完美期刊 — 先发表再追求顶刊
