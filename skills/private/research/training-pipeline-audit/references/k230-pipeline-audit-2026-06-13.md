## 一、项目概要

### 1.1 核心数据
- **K230帧数**: 901帧 (train=720, val=181)
- **OpenEDS帧数**: 976帧
- **图像尺寸**: 800×480 (K230) → 256×320 (训练)
- **标签格式**: JSON (all.json → train.json/val.json, 80/20)
- **Seg Mask**: 3类(0=bg, 1=iris, 2=pupil)

### 1.2 管线流程 (v6)
```
原图 → Step1(瞳孔定位) → Step2(生长+填充) → Step3(椭圆精修) 
     → Step4(ec标定) → Step5(3D能量蛇) → Step6(SAM眼裂+seg mask)
     → 训练(MBV2混合)
```

### 1.3 训练结果
- **模型**: MobileNetV2 + T3EM (35M参数)
- **训练策略**: 4阶段渐进 (P1=30, P2=20, P3=20, P4=20)
- **最佳结果**: Val Dice=0.8955, CErr=1.63px (epoch 89)
- **混合训练**: K230(901帧) + OpenEDS(976帧)

### 1.4 关键参数
| 参数 | 值 | 来源 |
|------|-----|------|
| ec | (434,142) | 7光斑法 |
| R | 380 | R = 2×r_iris |
| r_iris | 190 | 解剖参数 |
| D | 329 | √(R²-r_iris²) |

## 二、研究空白

### Gap 1: 单目3D眼球追踪精度理论极限未验证
- **现状**: 管线声称Val CErr=1.63px精度，但从未系统性量化误差来源
- **空白**: 单目摄像头+3D投影的精度瓶颈在哪里？是分割误差主导还是几何模型假设主导？
- **意义**: 如果分割误差贡献<50%的总误差，则几何模型是主要瓶颈；反之则分割优化优先

### Gap 2: SAM(深度学习)与传统CV(能量蛇/区域生长)性能边界未定义
- **现状**: 管线使用SAM生成眼裂mask，但仅用于Seg Mask，未系统性比较SAM vs 传统方法
- **空白**: 在什么条件下SAM优于传统方法？什么条件下传统方法更可靠？
- **意义**: 为虹膜分割算法选择提供实证指导

### Gap 3: 不同人种虹膜解剖学参数对分割精度的影响未评估
- **现状**: 管线固定ec=(434,142), R=380, r_iris=190为"解剖学约束"
- **空白**: 这些参数是否对所有种族/年龄/视力状况通用？
- **意义**: 临床部署前的泛化性验证

### Gap 4: 时序信息(16帧序列)对分割精度的增益未量化
- **现状**: K230_Dataset使用seq_len=16，但仅作为输入，未评估时序 vs 单帧
- **空白**: 利用时间连续性比单帧分析提升多少？
- **意义**: 为动态眼球追踪算法设计提供依据

## 三、科学假设

### H1: 分割误差主导单目3D眼球追踪总误差
- **Claim**: Val CErr 1.63px中，≥60%源于分割误差(而非几何模型)
- **Falsification**: 如果几何误差分析显示≥60%误差来自R/D/ec固定值，则H1被证伪
- **Test**: 系统性地消融几何模型固定参数，测量CErr变化

### H2: SAM在虹膜边缘分割上优于传统能量蛇，但在瞳孔中心定位上不如区域生长
- **Claim**: SAM在虹膜边缘Dice>0.92，区域生长在瞳孔中心定位CErr<1.0px
- **Falsification**: 如果SAM在瞳孔定位上CErr>1.5px，则H2被证伪
- **Test**: 对100帧图像同时运行SAM和能量蛇，分别评估瞳孔/虹膜Dice和CErr

### H3: 时序建模(16帧序列)比单帧分析提升Val Dice≥0.03
- **Claim**: 使用16帧序列的Val Dice比单帧提升≥3%
- **Falsification**: 如果单帧训练达到Val Dice≥0.88(接近序列的0.8955)，则H3被证伪
- **Test**: 训练单帧基准模型(无时序)，对比Val Dice

### H4: OpenEDS+K230混合训练对泛化性有显著增益
- **Claim**: 混合训练Val Dice比仅用K230训练提升≥0.05
- **Falsification**: 如果仅用K230训练达到Val Dice≈0.85，则H4被证伪
- **Test**: 仅用K230数据训练对照模型

### H5: 固定几何参数(ec/R/D)对分割精度有系统性偏差
- **Claim**: 固定ec=(434,142), R=380导致特定角度/特定人种的系统性分割偏差
- **Falsification**: 如果不同视角的分割精度无显著差异(p>0.05)，则H5被证伪
- **Test**: 按视角/光照分组分析Dice，进行ANOVA检验
