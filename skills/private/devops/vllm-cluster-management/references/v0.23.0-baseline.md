# vLLM v0.23.0 Release Notes — Current Baseline (Deployed Version)

**Source**: https://github.com/vllm-project/vllm/releases/tag/v0.23.0
**Date**: 2026-06-15
**Context**: This is the current deployed version on all three cluster nodes.

## Highlights

- 408 commits / 200 contributors (63 new)
- DeepSeek-V4 matured across backends
- Model Runner V2 expanded to Llama/Mistral dense models
- Rust frontend grew up (streaming generate, dynamic LoRA, /version, /server_info)
- Multi-tier KV cache offloading with object-store secondary tier
- Unified parser (reasoning + tool-call)

## Key Features for Current Stack

- MRv2 default for Llama and Mistral dense models (Qwen3 already on MRv2)
- Transformers v5 compatibility targets (vendor MiniCPM-V/O processors)
- Anthropic Messages API structured output support
- OpenAI Responses API with `system_fingerprint` field
- Security: SSL/TLS for data-parallel supervisor, untrusted-input hardening

## Deprecations in v0.23.0

- `JAISLMHeadModel` deprecated
- NixlConnector `kv_both` role deprecation cycle begun
- Functions from v0.21.0 deprecation scheduled for removal
