## 法向量（光轴）
n = [sin(a)*sin(b), -sin(a)*cos(b), cos(a)]
其中 a=arccos(b/a) 倾斜角, b=椭圆旋转角

## 光轴投影直线
cos(beta)*(x-x1) + sin(beta)*(y-y1) = 0

## 解剖约束
R_eyeball = 2*R_iris, d = sqrt(3)*R_iris
z1 = sqrt(3*R_iris^2 - (x1-x0)^2 - (y1-y0)^2)

## Rodrigues 旋转
v = O×O' (初始光轴×目标光轴)
M = I + [v]_x + [v]_x^2/(1+O·O')

## 完整流程
椭圆拟合 → 法向量 → 光轴直线 → 多帧交点=眼球中心 → Rodrigues→旋转矩阵→正交基→圆方程

## 陷阱
1. 单帧无法确定眼球中心
2. z1为虚数：椭圆参数与解剖约束矛盾
3. Center_eye在XY平面(z=0)
4. 多帧需迭代收敛
