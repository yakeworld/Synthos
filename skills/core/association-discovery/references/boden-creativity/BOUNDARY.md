# BOUNDARY.md — boden-creativity (Boden 创造力分类)

> 对应原则：P2

## 边界陈述

本模块的唯一职责：将研究空白/假设按其创新程度分类为组合/探索/转换创新。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| gap-type-classifier | GAP-TYPE 按内容分类空白 (方法/理论/实证/应用)；Boden 按创新度分类 |
| hypothesis-generation | 假设生成产生 H₁/H₂；Boden 评估 H₁ 的创新程度 |
| novelty_score in cases | 现有 novelty_score 是简单分数；Boden 提供理论基础和分类依据 |
| Kuhn/Lakatos | Kuhn/Lakatos 评估领域级范式；Boden 评估单篇假设的创新度 |

## 不包含

- 不做空白的发现
- 不做假设的生成
- 不做研究问题的形式化
- 不做期刊推荐 (仅提供参考)

## 输入/输出

- 输入：研究空白或假设描述
- 输出：JSON 格式的创新类型分类 + 创新度评分
