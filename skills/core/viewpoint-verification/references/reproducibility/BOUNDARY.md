# BOUNDARY.md — reproducibility (可重复性/开放科学框架)

> 对应原则：P0, P1

## 边界陈述

本模块的唯一职责：评估研究的可重复性和开放科学合规度。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| 实验设计 | 实验设计检查设计质量；本模块检查结果可复现性 |
| 伦理筛查 | 伦理检查合规性和风险；本模块检查方法透明性和可复现性 |
| Citation F1 | 引用 F1 检查引用完整性；本模块检查数据/代码可用性 |
| 4D 质量门控 | 4D 质量门做整体评分；本模块是"数据可靠性"维度的增强输入 |
| 可复现性检查 (P1) | 本模块直接满足 P1 原子可复现性要求 |

## 不包含

- 不做实验执行
- 不做数据分析
- 不做假设生成
- 不做论文写作

## 输入/输出

- 输入：论文全文或补充材料
- 输出：JSON 格式的可重复性评分 + Ioannidis 风险信号
