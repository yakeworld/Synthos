# BOUNDARY.md — meta-methodology (系统评价/Meta-分析方法论)

> 对应原则：P0, P1

## 边界陈述

本模块的唯一职责：检查系统评价/Meta-分析的方法学质量，包括 PRISMA 流程、质量评估工具、异质性分析、偏倚检测。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| 文献检索 (ACQ) | 文献检索负责搜索和筛选；本模块检查系统评价的方法质量 |
| 实验设计 (新增) | 实验设计检查原始研究设计；本模块检查综合研究 (Meta) 质量 |
| 4D 质量门控 | 4D 质量门做整体评分；本模块是"数据可靠性"维度的输入 |
| 引用 F1 | 引用 F1 检查引用完整性；本模块检查系统评价的引用/纳入逻辑 |

## 不包含

- 不做文献检索执行
- 不做原始实验设计
- 不做假设生成
- 不做论文写作

## 输入/输出

- 输入：系统评价/Meta-分析论文
- 输出：JSON 格式的 PRISMA 合规度 + 质量评估 + 偏倚检测 + 评分
