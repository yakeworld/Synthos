# BOUNDARY.md — experimental-design (实验设计方法论)

> 对应原则：P1

## 边界陈述

本模块的唯一职责：检查实验/研究设计的方法学质量（样本设计、随机化/盲法、对照、变量定义、数据分析）。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| CRISP-DM 实验流 | CRISP-DM 是数据处理流程 (Load→Preprocess→Model→Evaluate)；本模块检查实验设计本身 |
| 4D 质量门控 | 4D 质量门做整体评分；本模块输出作为"数据可靠性"维度的输入 |
| 伦理筛查 | Ethics 检查伦理风险和合规性；本模块检查方法学质量 |
| Pre-writing Gap Analysis | Pre-writing 在写作前规划实验；本模块在实验设计完成后评估 |
| 假设生成原子 | 假设生成产生 Hypothesis；本模块评估假设对应的实验设计 |

## 不包含

- 不做假设/研究问题的生成
- 不做数据收集或执行实验
- 不做论文写作
- 不做文献检索
- 不做结果解释

## 输入/输出

- 输入：实验设计描述/协议/论文 Methods 节
- 输出：JSON 格式的方法学质量评分 + 不合格项列表 + 修复建议
