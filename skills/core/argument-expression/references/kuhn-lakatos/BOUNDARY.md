# BOUNDARY.md — kuhn-lakatos (范式评估+研究纲领方法论)

> 对应原则：P2

## 边界陈述

本模块的唯一职责：评估假设所属领域的范式状态 (Kuhn) 和研究纲领健康度 (Lakatos)。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| Boden 创造力 | Boden 评估单篇假设的创新度；Kuhn/Lakatos 评估领域级范式和纲领 |
| Gap Type 分类 | GAP-TYPE 按内容分类；本模块按创新层次评估 |
| 4D 质量门控 | 4D 质量门做整体评分；本模块是"创新性"维度的输入源 |
| 假设生成原子 | 假设生成产生 H₁；本模块评估 H₁ 在领域中的位置 |

## 不包含

- 不做假设生成
- 不做单篇假设的详细论证分析 (Toulmin 的职责)
- 不做文献检索
- 不做实验设计

## 输入/输出

- 输入：假设/研究纲领描述
- 输出：JSON 格式的范式状态评估 + 研究纲领健康度
