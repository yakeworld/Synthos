# BOUNDARY.md — citation-chain (引用链分析)

> 对应原则：P2

## 边界陈述

本模块的唯一职责：补充 6 种引用功能分类，增加引用方向性 (前向/后向) 和网络结构分析。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| 引用功能分类 | 引用功能标注每篇文献的角色 (BG/SUP/CMP/GAP/METH/OBJ)；本模块补充方向性和网络维度 |
| 引用 F1 | 引用 F1 检查引用完整性 (P0/P1)；本模块检查引用网络质量 |

## 不包含

- 不做引用功能标注 (6种分类的职责)
- 不做引用数量统计
- 不做文献检索

## 输入/输出

- 输入：论文的参考文献列表 + 被引数据 (来自 Crossref/Semantic Scholar)
- 输出：JSON 格式的引用链分析 + 网络结构评分
