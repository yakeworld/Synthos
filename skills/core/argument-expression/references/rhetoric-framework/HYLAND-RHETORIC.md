# 科学写作的修辞学框架

> 理论来源：Hyland (2000) "Discourse Strategies"; Swales & Feak (2012) "Teaching Academic Writing"

## Hyland 学术写作五维度

Hyland 提出学术写作的评价框架，从 5 个维度分析论文写作质量：

### D1: Centrality (中心性) — 30%

```
定义: 论文是否将焦点保持在核心论点上，而非偏离。

检查项:
  - 每段是否有明确的主题句? (主题句在段首)
  - 段落是否围绕同一主题?
  - 是否有离题内容?
  - 过渡是否自然?

评分:
  所有段落聚焦核心 → 1.0
  偶有偏离但可拉回 → 0.6
  多次偏离 → 0.3
  严重偏离 → 0.0
```

### D2: Authority (权威建立/削弱) — 20%

```
定义: 作者如何通过语言建立/削弱学术权威。

权威建立:
  - 使用第一人称复数 "we show" (适度)
  - 谨慎措辞 "suggest", "indicate" (而非 "prove")
  - 引用权威文献支撑观点
  - 明确区分事实与推论

权威削弱:
  - 过度自信 "proves beyond doubt"
  - 无文献支撑的断言
  - 自我引用过多

评分:
  权威适当 + 区分事实/推论 → 1.0
  权威适当但略显不足 → 0.7
  过度自信 → 0.4
  无权威建立 → 0.0
```

### D3: Complexity (结构复杂度) — 15%

```
定义: 论证结构的层次和深度。

检查项:
  - 是否有多级论证 (主论点→子论点→证据)?
  - 是否有反方观点的处理?
  - 论证链条是否完整?
  - 逻辑连接词使用 (therefore, however, consequently)?

评分:
  多级论证 + 反方处理 + 完整链条 → 1.0
  单级论证但完整 → 0.6
  逻辑跳跃 → 0.3
  结构混乱 → 0.0
```

### D4: Expression (表达精确度) — 20%

```
定义: 语言表达的精确性和清晰度。

检查项:
  - 术语使用是否一致?
  - 是否有歧义表达?
  - 句子长度是否合理? (过长 > 40 字需检查)
  - 被动/主动使用是否适当?
  - 是否有语法错误影响理解?

评分:
  精确 + 一致 + 无歧义 → 1.0
  基本清晰但偶有模糊 → 0.7
  多处模糊 → 0.4
  理解困难 → 0.0
```

### D5: Introduction Strategy (引言策略) — 15%

```
定义: Introduction 的写作策略是否有效建立研究领域。

检查项:
  - 是否使用 CARS Model (Move1→Move2→Move3)?
  - 研究重要性声明是否有文献支撑?
  - 空白陈述是否具体?
  - 贡献声明是否可检验?
  - 论文路线图是否清晰?

评分:
  CARS 完整 + 所有要素优秀 → 1.0
  CARS 基本完整 → 0.7
  CARS 部分缺失 → 0.4
  无 CARS 结构 → 0.0
```

## Swales & Feak 学术写作风格矩阵

```
正式性 (Formality):
  高: 客观描述, 被动语态适度, 无口语
  低: 主观描述, 口语化表达

精确性 (Precision):
  高: 具体数字, 精确术语, 明确条件
  低: 模糊词汇, 笼统描述

简洁性 (Conciseness):
  高: 用最少的词表达清楚的意思
  低: 冗余, 重复, 不必要的修饰

评分标准:
  高 × 高 × 高 = 优秀 (1.0)
  至少 2 项高 = 良好 (0.7)
  1 项高 = 及格 (0.4)
  0 项高 = 需重写 (0.0)
```

## 不合格标准

```
1. D2 Authority 过度自信 + D1 Centrality < 0.5 → WARNING
2. D4 Expression 理解困难 → FAIL (需重写)
3. 写作风格矩阵 < 0.4 → WARNING
4. D5 Introduction Strategy 无 CARS → WARNING
```
