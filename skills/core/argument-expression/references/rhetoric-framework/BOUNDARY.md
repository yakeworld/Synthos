# BOUNDARY.md — rhetoric-framework (科学写作修辞学)

> 对应原则：P2

## 边界陈述

本模块的唯一职责：分析论文写作的修辞质量 (Hyland 五维度 + Swales & Feak 风格矩阵)。

## 与其他原子的边界

| vs 原子 | 边界 |
|---------|------|
| CARS 模型 | CARS 分析 Introduction 逻辑结构；修辞学分析 Introduction 写作策略 |
| Toulmin 论证 | Toulmin 分析单个论点逻辑；修辞学分析写作风格和表达质量 |
| IMRaD 验证器 | IMRaD 检查章节结构；修辞学检查章节内容质量 |
| Litreview 6 轴 | 6 轴评估文献综述质量；修辞学评估整体写作风格 |
| 4D 质量门控 | 4D 做整体评分；修辞学是"论证表达"维度的输入 |

## 不包含

- 不做逻辑分析 (Toulmin 的职责)
- 不做假设生成
- 不做文献检索
- 不做实验设计

## 输入/输出

- 输入：论文完整文本
- 输出：JSON 格式的 Hyland 五维度评分 + Swales & Feak 风格评估
